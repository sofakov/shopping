# Анализ покупательского поведения-датасета

## Описание проекта
Цель — провести анализ датасета аниме, выполнить визуализацию данных в Python и собрать дашборд в Yandex DataLens.

## Содержание проекта
- Анализ данных в Python (pandas, numpy, seaborn, matplotlib)
- Обработка пропусков, дубликатов, выбросов
- Визуализация данных (5+ графиков)
- Группировки и агрегации
- Дашборд в Yandex DataLens (3 визуализации + фильтры)
- Публикация проекта на GitHub

## Файлы
- `README.md` — Описание проекта
- `shopping_behavior.csv` — Исходный датасет
- `data_analysis.ipynb` — 1. Анализ данных в Python (pandas и numpy)
- 'https://datalens.yandex/vgtamtg32uduf' — 2. Аналитика в Yandex DataLens
